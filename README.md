# Skill Example Repository

Este repositorio es un ejemplo de una **Skill** para Antigravity.

## ¿Qué es una Skill?

Las Skills son carpetas de instrucciones, scripts y recursos que extienden las capacidades de Antigravity para tareas especializadas. Cada carpeta de skill contiene archivos que guían al agente sobre cómo realizar procedimientos específicos, utilizar herramientas particulares o seguir estándares de codificación determinados.

## Estructura Recomendada

Un repositorio de skill típico suele incluir:

- `SKILL.md`: (Obligatorio) El archivo de instrucciones principal con frontmatter YAML (nombre, descripción) e instrucciones detalladas en markdown.
- `scripts/`: Scripts de ayuda y utilidades que extienden las capacidades del agente.
- `examples/`: Implementaciones de referencia y patrones de uso.
- `resources/`: Archivos adicionales, plantillas o activos a los que la skill puede hacer referencia.

## Cómo usar este ejemplo

1. Explora el archivo `SKILL.md` (si está presente) para entender las instrucciones de la skill.
2. Revisa la carpeta `scripts/` para ver herramientas automatizadas.
3. Consulta `examples/` para ver cómo se aplica la skill en la práctica.
